Quran CLI

The Quran CLI is a command-line tool for exploring, annotating, and linking verses of the Quran. You can download the latest release from GitHub.

Resources

SQLite database

This is a precompiled SQLite database that the CLI pulls during installation.

Quran verses

This is a plain text file where each line is an Ayah from the Quran.

Quran pages

This CSV file contains pairs of Quran verse IDs pointing to lines in the Quran verses file. Each line represents a page in the Quran with 604 total pages. the first column is the starting verse on the page and the second column is the ending verse.

Chapers metadata

This is a CSV file organized as:

  • Ayah count
  • Ayah ID
  • Name
  • English name
  • Transliteration name

English translations

This is a plain text file where each line is an English translation of an Ayah from the Quran.
